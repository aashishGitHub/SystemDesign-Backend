📚 Caching Strategies Used:

1. Cache First: Check cache before network (for static assets)
cache → network → fallback

2. Network First: Try network, fallback to cache (for API calls)
network → cache → fallback

3. Stale While Revalidate: Return cache immediately, update in background
cache + network (background update)
💡 Key Concepts:
• Service Workers have their own lifecycle (install, activate, fetch)
• They can intercept network requests
• Work in a separate thread (like Web Workers)
• Enable offline-first applications
• Required for Progressive Web Apps (PWAs)
• Must be served over HTTPS (except localhost)

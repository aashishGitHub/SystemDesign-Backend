📤 File Upload Worker

Upload large files with chunking and progress tracking

🎯 What This Demonstrates:
This example shows how to use Web Workers to process large file uploads by splitting them into chunks. The worker handles the chunking logic, computes checksums, and simulates uploads without blocking the UI.

💡 How It Works:
1. File is selected by user
2. Worker splits file into chunks (1MB each)
3. Each chunk is processed and "uploaded" (simulated)
4. Worker computes MD5 hash for integrity
5. Progress updates in real-time
6. Main thread stays responsive!

🖼️ Image Processing Worker

Apply filters and effects to images using Web Workers

🎯 What This Demonstrates:
Image processing is CPU-intensive. By using a Web Worker, we can apply filters without freezing the UI. Upload an image and try different effects!

💡 How It Works:
1. Image data is extracted from canvas
2. Data is sent to worker via postMessage()
3. Worker applies filter to pixel data
4. Processed data is sent back and drawn
5. Main thread stays responsive throughout!
